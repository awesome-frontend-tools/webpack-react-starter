# Webpack React Starter

Template for your frontend webpack project

### Installation

```sh
npm install
```

### Start Dev Server

```sh
npm start
```

### Build Prod Version

```sh
npm run build
```

### Tests

```sh
npm run test:unit
```

### Features:

- Webpack (v5)
- TypeScript (v5)
- SASS Support
- LESS Support
- Eslint
- Prettier
- Jest (test, coverage)
- Bundle Analyzer (webpack-bundle-analyzer)
